fooskill
========

fooskill is a web app for managing foosball leagues and leaderboards!
The source code for fooskill is available at
`github.com/mwiens91/fooskill`_ and is licensed under the open-source
permissive `3-Clause BSD License`_.

If you find a bug in fooskill please raise an issue `here
<https://github.com/mwiens91/fooskill/issues>`_ or fix it yourself and
raise a pull request `here
<https://github.com/mwiens91/fooskill/pulls>`_ (but see
:ref:`contributing` first).
